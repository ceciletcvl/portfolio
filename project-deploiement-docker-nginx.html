Déploiement Docker + Nginx

Reverse proxy Nginx devant une app web, certificats auto (dev), healthchecks, logs, et mini-CI par hooks Git.

docker-compose.yml (extrait)

services:
  web:
    build: ./app
    restart: unless-stopped
    healthcheck:
      test: ["CMD", "curl", "-f", "http://localhost:5000/health"]
      interval: 30s
  nginx:
    image: nginx:alpine
    volumes:
      - ./nginx.conf:/etc/nginx/nginx.conf:ro
    ports: ["80:80"]
    depends_on: [web]

Observabilité

  • Access logs Nginx activés + rotation.
  • Healthcheck d’upstream (web) pour restart si besoin.
