Site d’inventaire réseau (Flask)

Application Flask pour suivre les équipements (IP, rôle, localisation), recherche, export CSV, scan ARP basique.

Fonctionnalités

  • CRUD équipements (Flask + SQLite).
  • Recherche et tri par IP/role.
  • Export CSV et import simple.
  • Scan ARP (subprocess) pour découvrir de nouveaux hôtes.

Snippet Flask (extrait)

from flask import Flask, render_template, request, redirect
import sqlite3
app = Flask(__name__)

def db():
    return sqlite3.connect("db.sqlite")

@app.route("/")
def home():
    q = request.args.get("q", "")
    con = db(); cur = con.cursor()
    if q:
        cur.execute("SELECT * FROM hosts WHERE ip LIKE ? OR role LIKE ?", (f"%{q}%", f"%{q}%"))
    else:
        cur.execute("SELECT * FROM hosts")
    rows = cur.fetchall(); con.close()
    return render_template("index.html", rows=rows, q=q)
